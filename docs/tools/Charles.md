## 代理转发请求

1、Tools > Map Remote

将 from 代理到 to

`import` 可以导入配置文件



2、安装证书

help > ssl proxying > install Charles Root Certificate

将登陆和系统的证书的 信任修改为 始终信任



3、Proxying > SSL Proxying setting

Include > location > host=* port=*
