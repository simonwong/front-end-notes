# 图

## 概念

图论（graph theory），是[组合数学](https://zh.wikipedia.org/wiki/组合数学)分支，和其他数学分支，如群论、矩阵论、拓扑学有着密切关系。[图](https://zh.wikipedia.org/wiki/图_(数学))是图论的主要研究对象。
