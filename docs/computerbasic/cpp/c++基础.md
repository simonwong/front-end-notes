#### 无符号类型与符号类型

**切勿混用无符号和有符号类型**

有符号会被转为无符号类型

```c++
int main () {
  unsigned u1 = 10, u2 = 42;
  int i = 10, i2 = -10;
  
  u2 - u1 // -> 32
  u1 - u2 // -> 4294967264
  
  u2 - i // -> 32
  i - u2 // -> 4294967264
    
  i2 + u1 // -> 4294967276 （i2 有符号类型会被转为无符号类型
}
```
